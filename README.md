# Wine-Quality-Prediction
End-to-end Wine Quality Prediction project using machine learning with an MLOps pipeline, Flask backend API, Streamlit frontend interface, and Dockerized deployment.

## Workflows
1. Update config.yaml
2. Update schema.yaml
3. Update params.yaml
4. Update the entity
5. Update the configuration manager in src config
6. Update the components
7. Update the pipeline
8. Update the main.py
9. Update the app.py

# How to run?
### STEPS:

clone the repository

```bash 
git clone https://github.com/arifaziz9162/Wine-Quality-Prediction
```

### STEP 01 - Create a conda environment after opening the reprository

```bash
conda create -n wine_ml python=3.11 -y
```

```bash
conda activate wine_ml
```

## STEP 02 - install the requirements

```bash
pip install -r requirements.txt
```
